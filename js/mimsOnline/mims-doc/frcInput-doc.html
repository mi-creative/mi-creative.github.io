Force Input Module



@{input_name} frcInput @{mass_to_send_force_to}

This allows to send an external force (audio rate signal) to a mass element inside the model. It must be named in[1,2,...] as this will define the corresponding inlet of the generated gen~ or Faust object for the force signal.

Example:
# Send force signal to a harmonic oscillator
@o1 osc 1 0.1 0.0.1 0. 0.8
@in1 frcInput @o1

Force Output Module

@{name} frcOutput @{observed_mass}
This allows to send the force buffer of a mass element inside the model to an output. It must be named in[1,2,...] as this will define the outlet number of the generated gen~ or Faust object.
Example:
# Listen to the force buffer (equivalent to acceleration) of a harmonic oscillator
@o1 osc 1 0.1 0.0.1 0. 0.8
@out1 frcOutput @o1

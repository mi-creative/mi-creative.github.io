Non-linear Bowing Module



@{name} nlBow @{m1} @{m2} {nlZ} {nlscale} {opt: smooth}

The bowing interaction is a non-linear linear interaction that allows stick-slip contacts between two material points. If the optional smooth parameter is unset or set to 0, a piecewise linear approximation is used (cf. blue curve). If smooth is set to 1, a smooth mathematical expression (red curve is used).

Physical Parameters: damping and distance scaling of the pluck.

Example:
@in1 posInput 0.
@o1 osc 1.2 0.1 0.001 0. 0.1
@nlb nlBow @in1 @o1 0.1 0.001

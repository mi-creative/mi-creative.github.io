Non-linear Plucking Module

@{name} nlPluck @{m1} @{m2} {stiffness} {nlscale} {damping}
The plucking interaction is a piecewise linear interaction that allows plucking contacts between material points (contact phase then release).
Stiffness and distance scaling of the pluck.
Example:
@in1 posInput 0.
@o1 osc 1.2 0.1 0.001 0. 0.1
@nlp nlPluck @in1 @o1 0.1 0.001

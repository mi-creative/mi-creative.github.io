Non Linear Spring Module

@{name} nlSpring @{m1} @{m2} {stiffness} {sq_stiff} {damping}
A visco-elastic spring between two material points, with a non linear (parabolic) component.
Linear stiffness, non-linear (parabolic) stiffness and linear damping
Example:
@g1 ground 0.
@m1 mass 1.2 0. 0.1
@sp nlSpring @g1 @m1 0.01 0.00001 0.001

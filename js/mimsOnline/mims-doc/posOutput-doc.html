Position Output Module


@{name} posOutput @{observed_mass}

This allows to send the position of a mass element inside the model to an output. It must be named out[1,2,...] as this will define the outlet number of the generated gen~ or Faust dsp object.

Example:
# Listen to the position of a harmonic oscillator
@o1 osc 1 0.1 0.0.1 0. 0.8
@out1 posOutput @o1

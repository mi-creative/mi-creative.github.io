MIMS-Online

Mass Interaction Model Scripter allows to generate, describe and tune 1D MI models with a simple scripting language.

MIMS is a simple graphical application, written in Python (using PyQt). Users can create 1D mass-interaction models by simply describing their topology, physical parameters, inputs and outputs. These descriptions can then be compiled into FAUST code (see mi-faust), or gen~ dsp code (see mi-gen).

Somewhere along the way, MIMS should be replaced by a clean and simpler web-based application. But for the brave of heart, install Python and start generating models !
